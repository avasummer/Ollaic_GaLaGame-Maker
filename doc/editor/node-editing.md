# 脚本与节点编辑

脚本以**指令流**呈现:每个节点是一张命令卡片,按顺序排列。点击节点在右侧 `DetailPanel` 编辑其字段。

## 指令流操作

| 功能 | 说明 |
|------|------|
| **插入节点** | 在任意位置插入(`InsertZone`);下拉按分类(剧情/场景/音频/控制/特效)选择命令类型 |
| **删除节点** | 从指令流移除,亦可在节点右键菜单删除 |
| **复制 / 剪切 / 粘贴** | 复制副本、剪切(删除+复制)、在任意位置粘贴;剪贴板在会话内保留 |
| **拖拽排序** | react-dnd 拖动节点重排,自动重连相邻节点 |
| **搜索 / 过滤** | 实时检索节点类型、角色名、内容、素材、配音文件,高亮匹配 |
| **选中与详情** | 点击节点选中,打开右侧详情面板编辑该类型专属字段 |

## 详情面板(按类型)

`DetailPanel` 根据节点类型渲染不同字段,例如:
- **对话**:角色名、对白、配音、立绘表情;
- **切换背景/立绘/小头像**:选择素材,立绘可设位置(左/中/右)、角色引用、表情、自定义 ID;
- **切换场景 / 调用场景**:用**场景下拉选择器**(按章节名展示、存精确文件名),见[场景管理](../scenes/scene-management.md);
- **选项分支**:增删选项,每项含文本与目标场景(同样用场景下拉);
- **音频(BGM/音效/视频)**:选文件 + 音量(0–100);
- **变量 / 用户输入 / 标签跳转 / 动画变换 / 解锁CG·BGM / 注释** 等,见[命令类型参考](./command-types.md)。
- **节点标志(flags)**:如 `-next`(立即执行下一条)、`-when`(条件执行)。

## 角色配色

为每个角色自动分配一个颜色,用于对话卡片、索引等处的视觉区分。
