# 功能文档 / Feature Docs

本目录按功能模块记录 **ciallo**(基于 WebGAL 的可视小说编辑器,Tauri + React)**当前已支持的功能**,用于提升可维护性与可读性。文档描述「做什么、对用户意味着什么、由哪些源码实现」,便于查阅与回溯。

> 前端源码位于 `design/src/app/`,Rust 后端位于仓库根的 `src-tauri` / `src`(通过 IPC `invoke` 调用)。各文档末尾的「相关源码」给出关键文件路径。

## 模块导航

| 模块 | 说明 | 目录 |
|------|------|------|
| **AI** | 对话式编辑 Agent、修改预览与应用、会话与记忆、AI 素材/立绘生成、供应商配置 | [`ai/`](./ai/README.md) |
| **编辑器** | 脚本/节点编辑、命令类型、脚本源码模式、撤销与快照 | [`editor/`](./editor/README.md) |
| **素材管理** | 素材库与分类、上传、元数据/别名、素材选择器 | [`assets/`](./assets/README.md) |
| **角色立绘** | 角色定义、立绘/表情系统、AI 立绘生成 | [`characters/`](./characters/README.md) |
| **场景关系与导航** | 场景关系图、场景增删改、章节/大纲元数据 | [`scenes/`](./scenes/README.md) |
| **项目与运行预览** | 项目创建/打开/导入/导出、运行预览、应用设置 | [`project/`](./project/README.md) |

## 约定

- 文档面向**功能**而非逐行实现;新增/改动功能时,请同步更新对应模块文档。
- 「相关源码」用于定位实现入口,具体行号会随代码漂移,以函数/组件名为准。
- 术语:**节点(node)** = 一条 WebGAL 指令;**场景(scene)** = 一个 `.txt` 脚本文件;**立绘(figure)** = 角色的人物图。
